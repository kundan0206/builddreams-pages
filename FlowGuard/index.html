PLATFORM

One product. Any controlled-workflow operation.

FlowGuard runs an Electron kiosk browser on every desk, monitors the OS for policy violations, syncs events to a FastAPI backend, and surfaces everything in an admin dashboard. The allowed URLs, detection rules, and form labels change per sector — the engine stays the same.

Works on Windows 10/11. No cloud dependency. Offline-first architecture.

Controlled access

Whitelist URLs, block new windows, disable DevTools, and keep users inside approved work tools only.

Proof detection

Detect on-page success states, extract reference numbers, and send verified completion events automatically.

Offline resilience

Cache events locally, retry sync on reconnect, and flag device health during network drops. No event is ever lost.

Fleet visibility

Live status of every device, branch, and operator. Alerts, SLA risk, and audit exports from one console.

COMMON QUESTIONS

Questions Indian buyers ask

Do we need to buy new hardware or replace existing PCs?
No. FlowGuard installs on existing Windows 10/11 desktops. We've deployed on machines as old as 2018 with 4GB RAM. No additional hardware purchase needed.
What happens if the internet goes down at a branch?
FlowGuard is offline-first. All events queue locally and sync when connectivity returns. No events are lost. The dashboard marks affected devices as "offline" and resumes automatically.
Is this compliant with RBI, NABH, or government audit requirements?
FlowGuard produces timestamped, tamper-evident event logs with operator ID, device ID, and reference numbers. These are exportable as PDF/CSV and have been used in RBI audits by NBFC customers and NABH accreditation by hospital chains.
How long does deployment take?
A standard 10–50 device rollout takes 14 days from contract signing to live. This includes configuration, URL whitelist setup, and admin training. No third-party integrations needed for basic deployment.
Where is our data stored? Can it be on our own server?
Yes. FlowGuard can be deployed on your own on-premise server, a private cloud, or AWS Mumbai / Azure India regions. We do not access your transaction data. Full DPDP Act 2023 compliance supported.
What's included in the 14-day free pilot?
Up to 10 devices, full software install, URL whitelist configured for your workflow, basic admin dashboard, and one walkthrough session. No invoice issued during the pilot. If it works, we convert to a paid plan. If not, you uninstall and walk away — no obligation.
