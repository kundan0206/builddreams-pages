SECURITY MODEL

Built for controlled environments, not casual browsing

The platform combines browser restrictions, operating-system process monitoring, local retry queues, signed device identity, and backend audit APIs. For each new sector, the implementation should validate the exact target website DOM and compliance requirements.

Kiosk controls

URL allowlists, blocked popups, disabled DevTools, navigation interception, and restricted menus.

Endpoint controls

Allowed process list, violation logging, heartbeat, local service API, and configurable policies.

Event integrity

Completion is triggered only after validated success text, reference extraction, and duplicate protection.

Admin governance

Device status, filings or events, alert review, evidence exports, operator reporting, and escalation workflows.
